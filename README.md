# Li CW 极简创作主题

一款为写作者、独立开发者、创作者设计的极简风格 WordPress 个人博客主题，极致性能、高度可自定义，无需冗余插件即可满足创作站点全部需求。

## 📦 安装方法

### 方法一：后台上传安装（推荐）
1. 下载主题压缩包 `li-cw-theme-vx.x.x.zip`
2. 进入 WordPress 后台 → 外观 → 主题 → 「添加新主题」→「上传主题」
3. 选择压缩包上传，点击「立即启用」

### 方法二：FTP 手动安装
1. 解压压缩包，得到 `li-cw-theme` 文件夹
2. 通过 FTP 上传至站点 `/wp-content/themes/` 目录
3. 后台 → 外观 → 主题，找到「Li CW 创作主题」并启用

## 🛠️ 基础配置

### 1. 基础页面设置
1. 后台 → 页面 → 新建两个空白页：「首页」和「日志」
2. 后台 → 设置 → 阅读 → 静态页面
   - 主页选择「首页」
   - 文章页选择「日志」
3. 保存更改

### 2. 导航菜单
1. 后台 → 外观 → 菜单
2. 创建新菜单，勾选「主导航」显示位置
3. 添加首页、日志、作品、关于、友链等页面到菜单

### 3. 可视化自定义
后台 → 外观 → 自定义，可配置全部内容：
- **首页设置**：欢迎语、大标题、副标题、描述、按钮、主视觉图
- **配色设置**：全站背景、文字、强调色、边框色自定义（支持 OKLCH）
- **字体设置**：标题/正文/界面/点缀四级字体栈自定义
- **关于页设置**：头像、昵称、身份、简介、社交链接
- **页脚设置**：版权信息、ICP备案号、隐私政策链接
- **友链设置**：通过后台「链接」菜单管理，数量无上限

### 4. 作品管理
后台左侧「作品」→ 添加新作品
- 标题：作品名称
- 特色图像：作品封面图
- 自定义字段 `li_cw_project_status`：作品状态（开发中/已完成等）
- 自定义字段 `li_cw_project_cat`：作品分类文案

### 5. 友链管理
后台左侧「链接」→ 添加链接
- 名称：站点名称
- 网址：站点链接
- 描述：站点一句话介绍
- 图片地址：站点头像/Logo

### 6. 代码高亮
- 文章中使用 WordPress 原生「代码块」粘贴代码即可
- 自动按需加载高亮脚本，无代码的页面不加载资源
- 自动生成行号、一键复制按钮，明暗模式自动适配配色

## ❓ 常见问题

### Q：「查看所有日志」点击没反应？
A：后台 → 设置 → 阅读，确认「静态页面」模式下已选择「文章页」对应的页面。

### Q：代码块没有行号/高亮？
A：确认使用的是编辑器「代码块」，不是「预格式化」块。

### Q：关于页头像不显示？
A：在「外观 → 自定义 → 关于页设置」中重新上传一次头像即可。

### Q：如何更新主题？
A：下载新版本压缩包，后台直接上传覆盖即可，自定义器配置不会丢失。
